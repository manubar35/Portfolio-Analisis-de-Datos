Proyecto SQL. Formada una base de datos desde 0 con varias tablas correctamente relacionadas entre si.
Proyecto PowerBI. Base de datos extraida de internet con una tematica similar a la creada en MySQL pero con una mayor cantidad de registros para 
poder realizar un analisis de datos mas profundo.
Proyecto Excel. Utilizando PowerPivot vemos resultados similares a PowerBi pero menos eficientes.
En ambos programas de Microsoft se utiliza el apartado de PowerQuery para una limpieza de datos.
Proyecto Python. Utilizado para consulta de datos como alternativa a SQL en archivos como excel o csv.
